DOM API

Challenge

Given the following HTML snippet:

<video id="myvideo">

<button>Play</button>
We have two distinct elements we want to connect together. The idea is when we click the button, we want to call video.play() method. With pure HTML and DOM APIs this is pretty simple:
  1. Put a click even listener on the button
  2. When invoked, query the DOM for the video element
  3. Invoke video.play()
Ember follows the Data-Down Actions-Up (DDAU) pattern, which isn't designed for something similar to a public DOM API. Still there are two ways to accomplish this functionality in ember:
  1. Lossly Coupled - the HTML way
  2. Strongly Coupled - the Ember way
