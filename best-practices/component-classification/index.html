Component Classification

Introduction

Based on Atom Design and the Single Element Pattern components can be classified as atoms, molecules, organism, primitives, shells up to elements. The approach for both systems is similar, the wording is different. I will use their approach and come up with a naming that works within ember and also connects to the patterns described here.

Classification

Atoms
The tiniest piecies, mostly primitive in their functionality.
Compound
Shallow containers whose purpose is to serve as building blocks to Elements.
Elements
Ready to use components that act in a similar fashion to html elements.
